# Project Feature Analysis (F1)

This document provides a comprehensive analysis of the features implemented in the "Viral Podcast Clip" project.

## 1. Authentication & User Management

- **Secure Authentication:** Implemented using `better-auth`, providing sign-in and sign-up functionalities.
- **User Profiles:** Persistence of user data (name, email, image) in a PostgreSQL database via Prisma ORM.
- **Credit-Based System:** Users start with a default of 10 credits, which are used to generate video clips.
- **Session & Account Management:** Support for user sessions and linking multiple OAuth accounts (e.g., Google, GitHub).

## 2. File Upload & Management System

- **S3 Integration:** Secure file uploads to AWS S3 using pre-signed URLs generated on the server.
- **Database Tracking:** Uploaded files are tracked in the database with detailed metadata (s3Key, displayName, status).
- **Status Lifecycle:** Files move through a state machine: `queued` -> `processing` -> `processed` / `failed` / `no credits`.
- **Unique Organization:** Files are organized in S3 using UUID-based directory structures to prevent collisions.

## 3. Viral Clip Generation Pipeline

- **Asynchronous Processing:** Triggered via server actions (`generateVideo`) and managed by Inngest background functions.
- **External Integration:** Communicates with a dedicated video processing endpoint to perform the actual clipping.
- **Automatic Discovery:** Background jobs automatically list generated clips in S3 and register them in the database.
- **Credit Deduction:** Dynamically deducts credits from the user's balance based on the number of viral clips successfully generated.

## 4. Clip Playback & Management

- **Secure Retrieval:** Generation of temporary S3 pre-signed URLs for secure clip playback in the browser.
- **Relational Integrity:** Clips are linked to both the original `UploadedFile` and the `User` for efficient querying.

## 5. Dashboard & UI/UX

- **Unified Dashboard:** A central interface for users to monitor their uploads and access generated clips.
- **Modern Tech Stack:** Built with Next.js 16 (App Router), Tailwind CSS (v4), and Shadcn UI for a polished, responsive experience.
- **Real-time Feedback:** Visual status indicators for the video processing pipeline.

## 6. Infrastructure & Tooling

- **Background Jobs:** Inngest for reliable, event-driven background processing.
- **Database ORM:** Prisma for type-safe database access and migrations.
- **Payments Ready:** Stripe integration (dependency present) for future credit purchasing features.
